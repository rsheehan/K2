Starting a new slide
    state : ST_START

state stays ST_START while reading commands e.g. "#SLIDE"
When the slide separator "#-..." is reached the state changes to ST_TITLE
Stays ST_TITLE until the closing slide separator "#-..."
Then the state changes to ST_CONTENT.

State then changes to ST_START if a command is reached.
Or state changes to ST_TITLE if a new slide separator "#-..." is reached.
